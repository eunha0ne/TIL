# 네트워크

## LB(로드 밸런싱)과 CNAME

`DNS(Domain Name System)`를 이용해서 커스텀 도메인을 사용할 때, 서브 도메인을 등록하기 위해서는 `CNAME` , `A` 레코드 등의 정보를 DNS 서비스 홈페이지에 등록해야 한다. DNS 서버에는 도메인 주소와 IP 주소가 한 쌍으로 저장되어 있다. 이 하나의 행을 `Record(레코드)`라고 부르며, 저장되는 형태에 따라서 타입이 구분된다.
